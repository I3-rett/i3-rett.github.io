## Let's go !

Even if the tutorial is for Blender 3.3, I will use the version 4.4.3 (latest at the time of writing). 

## The cannon tower 

I needed to install an add-on to be able to create a *round cube*.

In Edit > Preferences > Get Extensions, search for *Extra Mesh Objects* and enable it.

Then, in the 3D viewport, press `Shift + A` to open the add menu and select *Mesh > Round Cube*.
